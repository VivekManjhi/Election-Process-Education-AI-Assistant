
# 🗳️ Election AI Assistant

A simple and interactive AI chatbot that helps users understand election-related queries like voting process, voter ID, and eligibility.

---

## 🚀 Live Demo
👉 https://sunny-starship-de828a.netlify.app

---

## 📌 Features

- 💬 Chat-based user interface  
- 🗳️ Provides information about voting  
- 🪪 Helps with voter ID related queries  
- ⚡ Fast and responsive UI  
- 🎯 Beginner-friendly project  

---

## 🛠️ Tech Stack

- HTML  
- CSS  
- JavaScript
